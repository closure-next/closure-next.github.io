Closure Next Bot Privacy Policy

Last Update: 02/01/2025

At Closure Next, we value your privacy and are committed to ensuring transparency regarding the data collection and use of our services, including the Closure Next Bot. This Privacy Policy outlines how the bot operates and explains what data, if any, it collects.

1. Data Collection:

The Closure Next Bot does not collect any personal data from its users. The bot simply operates as a bridge between GitHub and Telegram to share updates about changes in the Closure Next repository.

When a change is made in the Closure Next repository (e.g., a commit or pull request), the bot is notified through GitHub's API. It then makes a simple API call via curl to the bot's repository to post a message in the Telegram channel. This message includes the following details:

- The name of the user who made the change (the committer).

- The message written by the user, which could be the commit message or pull request description.

This is the only information the bot processes. The bot's purpose is solely to organize and display changelogs for the Closure Next project, ensuring that everyone can view updates to the repository.

2. APIs Used:

The Closure Next Bot interacts with two primary APIs:

- GitHub API: The bot retrieves information related to changes made in the Closure Next repository, including commits and pull requests.

- Telegram API: The bot publishes messages to the Closure Next Telegram channel about the changes made in the repository.

No personal information is collected or stored by the bot; it only processes the necessary data related to repository changes.

3. Open Source:

The Closure Next Bot is an open-source project, and the full source code is publicly available. You can review the code to verify its operation and ensure no personal data is being collected or used inappropriately.

You can access the bot's repository on GitHub at:

https://github.com/closure-next/bot

The bot is licensed under the MIT License, allowing anyone to inspect, modify, or contribute to the project.

4. Security:

The Closure Next Bot follows best practices for security, ensuring that any interactions with external APIs are secure and do not expose sensitive information. However, we recommend users to review the GitHub and Telegram privacy policies for more information on how they handle data.

5. Changes to this Policy:

This Privacy Policy may be updated from time to time to reflect changes in the way the bot operates or in response to changes in legal or regulatory requirements. We encourage you to review this policy periodically to stay informed.
