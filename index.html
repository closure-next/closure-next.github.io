What is Closure Next?

Closure Next is the Closure Library rewritten in TypeScript, with native integrations with several popular frameworks, Angular, Cypress, Deno, Jest, Next.js, Node.js, Nuxt.js, React, Rollup, Svelte, Vitejs, Vuejs, WebAssembly, Web Components and Webpack, making this new version of the Closure Library a Full-Stack tool.
